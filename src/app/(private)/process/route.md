# Process Module Routes

## Main Routes

- `/process` - List of all processes (index page)
- `/process/[id]` - Process detail page with tabs (Resumo, Atividades, Histórico)

## Tabs Structure

### Resumo (Summary)
- Process metadata (CNJ, parties, class, court)
- Quick actions (edit, export, print)

### Atividades (Activities)
- Deadlines list with filters
- Documents list with search
- Create new activities

### Histórico (Timeline)
- Timeline of events (external/internal)
- Chronologically ordered
- Mark as viewed functionality
